# NCERT AI Assistant

An AI-powered assistant designed to help with NCERT (National Council of Educational Research and Training) content. This project aims to provide intelligent support for educational materials, question answering, and learning assistance based on NCERT curriculum.

## Features

- Intelligent question answering based on NCERT content
- Educational content analysis
- AI-driven learning recommendations
- Support for multiple subjects and grades

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Sushildevelop/NCERT_AI_ASSISTANT.git
   cd NCERT_AI_ASSISTANT
   ```

2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Run the application:
   ```bash
   python main.py
   ```
